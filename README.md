<h3>¿Qué es un lenguaje de programación?</h3>
Es un sistema de comunicación que permite darle instrucciones a una computadora para que realice tareas. Sirve para procesar información, crear aplicaciones, páginas web y juegos. 
Funciona escribiendo instrucciones siguiendo la sintaxis del lenguaje. 
Ejemplo en Python

```python
nombre = "María"
color = "rosado"
print("Mi nombre es" + nombre + "y mi color favorito es" + color)
```

Existen muchos lenguajes de programación, entre ellos se encuentran Python, JavaScript, Java, C++.

<h3>¿Qué es un algoritmo?</h3>
Un algoritmo es una serie de pasos ordenados que permiten resolver un problema o realizar una tarea. 

<h3>Variable</h3>
Es un espacio en memoria donde se almacena un dato, su valor puede cambiar durante la ejecución del programa. Tiene un nombre y un valor, sirve para guardar información que el algoritmo necesita. 

<h3>Funciones</h3>
Una función es un bloque de código que realiza una tarea específica, permite agrupar instrucciones y reutilizarlas, puede recibir datos llamados parámetros. Puede devolver un resultado o simplemente ejecutar acciones.

<h3>Constantes</h3>
Una constante es un valor que no cambia durante la ejecución del programa. Se usa para guardar información fija y que debe mantenerse igual. Tiene un nombre y un valor, igual que una variable.
Ayuda a evitar errores y hacer el código más claro.

<h3>Reflexión</h3>
Aprender sobre algoritmos, variables, funciones, me está permitiendo entender que la programación no es solo escribir código, sino pensar de forma lógica y ordenada. Cada concepto cumple un papel importante, las variables y constantes almacenan datos, las funciones organizan las acciones, y el algoritmo define los pasos para resolver un problema. Con la práctica, he descubierto que crear un programa es darle solución a un problema, donde cada linea tiene un propósito.
